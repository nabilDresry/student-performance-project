# Prediksi Prestasi Akademik Siswa

## Deskripsi

Proyek ini bertujuan untuk Memprediksi Prestasi Akademik Siswa Berdasarkan Faktor Akademik, dan Aktivitas Non-Akademik Menggunakan Metode Supervised Learning.

## Dataset

* Jumlah Data: 2392 baris
* Jumlah Fitur: 15+
* Target: GPA

## Algoritma

1. Linear Regression
2. Random Forest Regression

## Evaluasi

Model dievaluasi menggunakan:

* MAE
* MSE
* RMSE
* R²

## Deployment

Aplikasi dideploy menggunakan Streamlit Cloud dan terhubung dengan GitHub Repository.
